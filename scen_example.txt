# Сценарий прогона


# Кроме имени пользователя для теста можно указать значение 'take_from_testname', это означает,
# что имя пользователя берется из открывающего тэга теста после знака '%' (из '#!$Все_роли%Кассир' будет взят 'Кассир')
# Вместо имени файла доп. теста можно указать 'only_start' - тогда программа автоматически сформирует шаг запуска эталона
# При замене строки параметры отделяются друг от друга запятой. Первый параметр заменяемая строка, второй строка на которую заменяют.
# При замене строки второй параметр может быть равен значению '@base_addr' это значит, что первый параметр изменится на адрес развернутой базы с текущим эталоном на раннере.
# При замене строки количество параметров должно быть четным.
# В настройках каждого теста параметр ЭТАЛОН должен идти всегда первым. 
 
# Данный пример запускает два одинаковых теста подряд
